# AKS Module

This module deploys an **Azure Kubernetes Service (AKS)** cluster.

---

## Purpose

Provides a standardized and configurable Kubernetes platform for workload landing zones.

---

## Resources created

- Azure Kubernetes Service cluster
- System node pool

---

## Configuration capabilities

- Optional deployment via enable flag
- Configurable Kubernetes version
- Adjustable node pool sizing
- Autoscaling support
- Tag inheritance from landing zone

---

## Extensibility

Future enhancements may include:
- Additional node pools
- Private clusters
- Private endpoints
- Ingress controllers
- GitOps integration
